# AutoProduct — Landing Page (Next.js + Tailwind)

A responsive landing page built with Next.js and Tailwind CSS for the AutoProduct demo.

## Features
- Hero (heading, subtext, CTA)
- Features (3 items)
- Testimonials (3 cards)
- Contact Section (form + contact details)
- Footer
- Fully responsive and accessible base structure

## Tech
- Next.js
- Tailwind CSS

## Run locally
1. `npm install`
2. `npm run dev`
3. Open `http://localhost:3000`

## Vercel v0 usage
I used Vercel v0 to prototype the Hero, Features, and Testimonials UI variants — exported markup and adapted to Tailwind.
